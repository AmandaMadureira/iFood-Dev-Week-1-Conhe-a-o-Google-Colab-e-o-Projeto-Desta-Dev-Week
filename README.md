# Projeto - Cálculo de NPS com Python

Este projeto foi desenvolvido durante a mentoria **iFood Dev Week #1** com o objetivo de praticar conceitos de Python, análise de dados e programação orientada a objetos.

## Tecnologias utilizadas

* Python
* Pandas

## Funcionalidades

* Leitura de arquivos CSV
* Cálculo de NPS (Net Promoter Score)
* Classificação de promotores e detratores
* Implementação procedural
* Implementação orientada a objetos

## Estrutura do projeto

### Versão 1 - Procedural

Utiliza funções para calcular o NPS a partir das notas presentes no arquivo CSV.

### Versão 2 - Orientada a Objetos

Implementa classes `Feedback` e `AnalisadorFeedback`, aplicando conceitos de encapsulamento e organização de código.

## Objetivo

Praticar:

* Manipulação de dados com Pandas
* Estruturas condicionais
* Classes e objetos em Python
* Boas práticas de organização de código

## Como executar

1. Instale as dependências:

```bash
pip install pandas
```

2. Execute o arquivo Python:

```bash
python nome_do_arquivo.py
```

3. Certifique-se de que o arquivo `teste.csv` esteja no mesmo diretório do projeto.
